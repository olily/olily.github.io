1144 The Missing Number (20分)

Directions:

Given N integers, you are supposed to find the smallest positive integer that is NOT in the given list.

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer N (≤105). Then N integers are given in the next line, separated by spaces. All the numbers are in the range of int.

Output Specification:

Print in a line the smallest positive integer that is missing from the input list.

Sample Output:
分析：
  1. 给定n个int型数据，要求找出其中未出现的最小的正整数。
  2. 本题给定范围是N<=105,所以理所当然最,小正整数一定<=105+1，所以可以采用木桶思想，开一个visit数组记录出现过的正数记为true，最后遍历visit数组，遇到的第一个为false的正数即为结果。
  3. Note（本题有几个点需要注意）:
    • 注意数据的范围（当然其实开到最大也没问题，就是有点浪费空间）；
    • n的范围是0-105，而最小正整数的范围其实要比n大1
    • 虽然n的范围是<=0-105，但每一个数据却比一定在这个范围，所以在记录visit的时候，需要先判断该数据是否在定的范围内，否则会出现段错误（尤其是最后一组测试数据）；
