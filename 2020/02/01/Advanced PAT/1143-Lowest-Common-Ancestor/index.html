1143 Lowest Common Ancestor (30分)

Posted on 2020-02-01 Edited on 2020-02-02 In Advanced PAT

笑一笑，十年少

Directions:

The lowest common ancestor (LCA) of two nodes U and V in a tree is the deepest node that has both U and V as descendants.

A binary search tree (BST) is recursively defined as a binary tree which has the following properties:

The left subtree of a node contains only nodes with keys less than the node’s key.
The right subtree of a node contains only nodes with keys greater than or equal to the node’s key.
Both the left and right subtrees must also be binary search trees.
Given any two nodes in a BST, you are supposed to find their LCA.

Input Specification:

Each input file contains one test case. For each case, the first line gives two positive integers: M (≤ 1,000), the number of pairs of nodes to be tested; and N (≤ 10,000), the number of keys in the BST, respectively. In the second line, N distinct integers are given as the preorder traversal sequence of the BST. Then M lines follow, each contains a pair of integer keys U and V. All the keys are in the range of int.

Output Specification:

For each given pair of U and V, print in a line LCA of U and V is A. if the LCA is found and A is the key. But if A is one of U and V, print X is an ancestor of Y. where X is A and Y is the other node. If U or V is not found in the BST, print in a line ERROR: U is not found. or ERROR: V is not found. or ERROR: U and V are not found..

分析：
  1. 本题是背景是在搜索树（BST）的基础上给定先序遍历，要找到给定两个节点的最小公共祖先节点。
  2. 两个思路：
    • 简单粗暴款：BST伴随一个隐含条件，即该树的中序遍历是升序，再结合题中给出的先序序列，所以可以确定树的结构；然后再比较两个节点的路径，其中数值相同且最小（不是指值最小）的节点即为最小公共祖先节点。（代码写到一半给否了，因为过程实在是太繁琐了，而且在最后一步确定路径那里不太好处理，所以有兴趣的小可爱就自己探索吧~奥利给）
    • 神仙思路款：因为测试用例包含了判断所给节点是否存在的情况，所以用一个map型的<int,bool>结构可以节省很多时间（不用bool型的一维数组是因为测试用例的值有可能超出数组下标的范围，比如负数、大数），并且自带升序结构还可以起到一个伪中序的作用。要找u和v的公共祖先，由BST的定义可以知道，任意一个节点的左子树一定小于该节点的data，而右子树节点的值均大于该data,而且在先序遍历中，该节点一定会出现在u和v的前面。所以只需要遍历先序序列，然后找到一个节点的值a且范围在[u,v]之间即可跳出遍历，若a=u和v中其中之一，则说明与a相等的节点为祖先节点，另一节点为子孙节点（思路来自柳神，附上柳神主页传送门：https://www.liuchuo.net/ ）。
  3. Note：
    • 用第二种思路简直是如有神助啊~不过考试的时候如果以前没有考虑过那应该是想不到的；
    • 保持学习。
