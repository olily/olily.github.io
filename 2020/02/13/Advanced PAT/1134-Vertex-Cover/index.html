Directions:

A vertex cover of a graph is a set of vertices such that each edge of the graph is incident to at least one vertex of the set. Now given a graph with several vertex sets, you are supposed to tell if each of them is a vertex cover or not.

Input Specification:

Each input file contains one test case. For each case, the first line gives two positive integers N and M (both no more than 100000, being the total numbers of vertices and the edges, respectively. Then M lines follow, each describes an edge by giving the indices (from 0 to N−1) of the two ends of the edge.

Output Specification:

分析：
  1. 给定一个图：包含n个顶点和m条边。然后再给出k个集合，其中每个集合包含Nv个顶点。要求对每一个集合进行判断：判断每一条边是否至少有一端在该集合内，若满足则输出Yes，否则输出No。
  2. 使用edge结构体保存每一条边的两端，定义一个vectorg(m)保存所有边，然后对每一个结合进行判断；定义一个bool型数组s保存集合中出现的顶点编号，然后对边集进行遍历，若两个存在一条边的两个端点v1、v2其在s中都为假，则退出循环输出No,否则输出Yes。
  3. Note：其实本题要实现这个算法是很简单的，但是关键是有时空限制，若定义二维数组则会超出空间限制，若每一个集合的每一个顶点都遍历一次边集则超时。
